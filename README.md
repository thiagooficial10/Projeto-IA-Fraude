# Projeto de IA para Detecção de Fraude
Este projeto simula transações financeiras e utiliza Inteligência Artificial para identificar fraudes.

## Etapas do Projeto:
1. **Geração de Dados:** Criação de base simulada com características realistas.
2. **Envio ao Banco:** Integração com PostgreSQL (NeonDB) via Python.
3. **Análise e Modelagem:** Treinamento de modelo para detectar fraudes.

## Tecnologias Utilizadas:
- Python
- Pandas / Numpy
- PostgreSQL
- GitHub
